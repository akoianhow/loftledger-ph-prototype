# LoftLedger PH Prototype

This is a self-contained static prototype for a Filipino pigeon loft organizer.

## How to open

Open `index.html` in any modern browser. No install, server, account, or database is required.

## Included sample data

- 4 seeded birds
- At least 2 related records per bird
- 5 seeded training sessions with release points, distance, weather, arrival gaps, and late/rested statuses
- 3 seeded races with club, release point, basketing date, entry fee, prize pool, entries, clocking, rank, velocity, and winnings
- Health, breeding, training, race, reminder, and expense examples
- PHP-based expense records
- Local Philippine release/source context

## Prototype interactions

- Search by ring number, name, strain, source, or note
- Filter by status and sex
- Click a bird card to inspect profile, pedigree, and records
- Switch record tabs in the detail panel
- Review Race Center entries and race result tables
- Review per-bird training readiness, return rate, average gap, late returns, rest days, and race conversion
- Save temporary browser-only handler notes
- Export the sample data as JSON

## Scope

This is not a production app yet. It is meant for feedback from fanciers before deciding the real database, user roles, mobile workflow, and reporting needs.
